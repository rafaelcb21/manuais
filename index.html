Instalar NodeJs, Express, Angular 2, Couchdb, Ionic 2 e Pounchdb em Produção

Passo 1: Instalar NodeJs

Entre no site https://nodejs.org/en/ faça o download da versão corrente (ex: v.6.2.2 Current) e instale o NodeJs.

Passo 2: Instalar Express

Instale o Express de forma global com o comando abaixo:

npm install express-generator -g

Após instalado o Express, crie a aplicação, que por enquanto, não terá o Angular 2.

express myApp --hbs

A opção hbs significa Handlebars Template (Controle, Guia, Direcionamento de Templates). HBS é um sistema semântico de templates, na qual esta relacionado a extensões como Html e Js. Algumas notações nos templates {{ }}, [ ], ( ), [ ( ) ] Ao escolher a opção --hbs o Express não usará o Jade, que é default na instalação.
